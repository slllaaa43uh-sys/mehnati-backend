- **أول 10 جيجابايت تخزين مجاني دائماً**
- لا حاجة لبطاقة ائتمان لإنشاء الحساب

## تسعير Pay-As-You-Go
- **التخزين**: $6 لكل تيرابايت شهرياً
- **التحميل (Egress)**: مجاني حتى 3x من متوسط التخزين الشهري
- **التحميل الإضافي**: $0.01 لكل جيجابايت

## المميزات
- لا توجد رسوم على الحد الأدنى لحجم الملف
- لا توجد رسوم على الحد الأدنى لمدة التخزين
- متوافق مع S3 API
- Class A API calls مجانية
- Class B و C API calls: أول 2,500 يومياً مجانية
